# perX UI/UX Reference Analysis

The complete per-file map is maintained in `docs/UI_REFERENCE_PAGE_MAP.md`.

## Primary Patterns Extracted

- Desktop app: fixed navy sidebar, fixed top header, content scroll container, three-column dashboard composition and right-side activity/details rails.
- Mobile app: compact navy top bar, visible search, stacked priority content, padded horizontal scrollers and fixed bottom navigation with safe-area spacing.
- Light mode: off-white page background, white cards, navy navigation, blue primary actions, purple selected accents and subtle shadows.
- Dark mode: charcoal/navy page canvas, layered dark panels, purple selected states, blue secondary actions and high-contrast text.
- Messaging: conversation list, active conversation, context/deal details and composer as a professional workspace.
- Discover: search-led interface with type tabs, filters, result cards, trust markers and pagination.
- Escrow/deals: security-led panels, process steps, status tables and transaction summaries.

## Design Decisions

- The current black-and-gold theme was overridden because the approved references use navy, off-white, blue, purple and charcoal as the dominant system.
- Gold remains for the official logo, trust/verification details and occasional badges.
- UI screenshots are used only as visual references, not as content imagery.
- perX ecosystem content remains broader than freelance or property-only workflows.
